###### 查看分支
```shell
git branch
```
###### 修改本地分支名
```shell
git branch -m oldName newName
```
###### 切换分支
```shell
git checkout xxxx
```
###### 创建本地分支,并且切换到分支
```shell
git checkout -b xxxx
```

##### 删除本地分支
```shell
git branch -d xxxx
```
##### 删除远程分支
```shell
git push origin --delete xxxx
```

#### 拉取分支
```shell
git pull
```
